WebP Converter

Локальный GUI-конвертер изображений в WebP с интерактивной обрезкой и массовым переименованием.

GitHub репозиторий Быстрый старт

Возможности

  • Batch-конвертация JPG/PNG/TIFF/BMP → WebP
  • Интерактивная обрезка (с пресетами соотношений сторон)
  • Пресеты качества + ручной режим
  • Умное массовое переименование (генератор уникальных имён)
  • Полностью локальная обработка (ничего не загружается в интернет)

Запуск

pip3 install -r requirements.txt
python3 webp_converter.py
