# WebP Converter

Локальный конвертер изображений в WebP с интерактивной обрезкой.

## Возможности

- ✅ Конвертация JPG, PNG, JPEG, TIFF, BMP → WebP
- ✅ Интерактивная обрезка с предпросмотром в реальном времени
- ✅ Batch-режим (обработка нескольких файлов)
- ✅ Drag & Drop загрузка файлов
- ✅ Пресеты качества (SEO, Balanced, High)
- ✅ Пресеты соотношений сторон (8:5, 16:9, 4:3, 1:1)
- ✅ Умное переименование файлов (SEO-оптимизация)
- ✅ Сохранение настроек между сессиями
- ✅ Предпросмотр изображений
- ✅ Тёмная тема интерфейса
- ✅ 100% локальная обработка без загрузки в интернет

## Быстрый старт (запуск из исходников)

```bash
cd webp_converter
pip3 install -r requirements.txt
python3 webp_converter.py
```

## Сборка .app для macOS

Чтобы создать полноценное приложение с иконкой:

```bash
cd webp_converter
chmod +x build_mac.sh
./build_mac.sh
```

После сборки:
1. Откройте папку `dist/`
2. Перетащите `WebP Converter.app` в `/Applications`
3. Готово! Теперь можно запускать двойным кликом

### Добавление своей иконки

1. Создайте иконку в формате `.icns` (можно конвертировать из PNG)
2. Положите файл `icon.icns` в папку проекта
3. В `build_mac.sh` замените `--icon NONE` на `--icon icon.icns`
4. Пересоберите

## Использование

### Простая конвертация

1. Перетащите изображения в окно приложения (или кликните для выбора)
2. Выберите качество (пресет или вручную)
3. Нажмите **Конвертировать**
4. Готовые файлы сохраняются в папку `converted/`

### Обрезка + конвертация

1. Добавьте изображение
2. Дважды кликните на нём в списке (или выберите и нажмите **Обрезать**)
3. В окне обрезки:
   - Выберите соотношение сторон (Обложка 8:5, 16:9 и др.)
   - Перетаскивайте рамку мышью
   - Введите точные размеры в полях Ширина/Высота
4. Нажмите **Применить**
5. Конвертируйте как обычно

### Массовое переименование (SEO)

1. Добавьте файлы
2. Нажмите **Переименовать**
3. Введите ключевые слова через пробел: `december turkey`
4. Система автоматически сгенерирует уникальные имена:
   - `december-turkey.webp`
   - `turkey_december.webp`
   - `beautiful-december-turkey.webp`
   - и т.д.
5. Нажмите **Применить**

### Пресеты качества

| Пресет | Quality | Применение |
|--------|---------|------------|
| SEO (70) | 70 | Для веб-страниц, максимальная экономия |
| Balanced (75) | 75 | Оптимальный баланс размер/качество |
| High (85) | 85 | Для контента с высокими требованиями |

### Пресеты соотношений сторон

| Пресет | Соотношение | Пример |
|--------|-------------|--------|
| Обложка 8:5 | 8:5 | 1920×1200 |
| Широкий 16:9 | 16:9 | 1920×1080 |
| Стандарт 4:3 | 4:3 | 1600×1200 |
| Квадрат 1:1 | 1:1 | 1000×1000 |

## Решение проблем

### Приложение не запускается после сборки

```bash
# Разрешите в Системных настройках → Безопасность
# Или запустите через терминал:
open "dist/WebP Converter.app"
```

### "No module named PySide6"

```bash
pip install PySide6
```

### Pillow не поддерживает WebP

```bash
pip uninstall Pillow
pip install Pillow --no-cache-dir
```
